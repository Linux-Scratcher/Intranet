# Intranet
Simple Intranet for youre local network<br>
# How to<br>
Clone repository
`git clone https://github.com/Linux-Scratcher/Intranet.git`<br>
Go in the folder
`cd Intranet`<br>
Run
`python3 script.py`<br>
Go to url
`http://localhost:8084/`<br>
